EvaEvents API

Objects:

  • Organizations - /api/organizations
  • Forums - /api/forums
  • Events - /api/events
  • News - /api/news
  • Persons - /api/persons
  • Rooms - /api/rooms
  • Likes - /api/likes
  • Event types - /api/event_types
  • Colors - /api/colors
  • Tasks - /api/tasks

Arguments Examples:

  • Sorting:
    • /api/events?sort=id_asc
    • /api/events?sort=updated_at_desc
  • Limit - /api/events?limit=2

Forum inherited objects

  • GET all forum Events - /api/forums/1/events
  • GET all forum Rooms with near event for every room - /api/forums/1/rooms?near_event=1&cur_date=2015-08-14T23:00:00.000Z
  • POST or DELETE Person by id as visitor to forum - /api/forums/1/persons?role=visitor&id=12

Personal requests(for current authorized user)

  • GET personal News - /api/forums/1/news
  • POST or DELETE Like to some of Forum inherited objects - /api/forums/1/news/2/likes
  • POST or DELETE Subscribe to some Event - /api/forums/1/events/2/subscribes
  • GET personal subscribed Events - /api/forums/1/persons/2/subscribed_events

Authenticate

  • Anonymous register(no need arguments, return id and password) - /api/anonym_register
  • Login(need id and password) - /api/login
  • Logout(no need arguments, just remove from session) - /api/logout
  • Current user id - /api/current_user_id
